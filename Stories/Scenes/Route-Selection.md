# Route Selection

Three representatives present incompatible plans in the same transit lounge. The Ministry wants restoration, Freeplot wants distributed stewardship, and the Studio wants a spectacular reboot.

The routes reconverge because all three need the same four credentials, but each alliance changes Tess's available final arguments.
